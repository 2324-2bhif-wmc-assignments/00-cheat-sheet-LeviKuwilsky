HTML elements

Tags

  1. <html> - Defines the root of an HTML document
  2. <body> - Defines the document's body
  3. <h1> - to <h6> Defines HTML headings
  4. <br> - defines a line break

    <a href="https://www.example.com" title="Visit Example Website">Link to Example</a>

Attributes

Summary

  1. All HTML elements can have attributes
  2. The href attribute of <a> specifies the URL of the page the link goes to
  3. The src attribute of <img> specifies the path to the image to be displayed
  4. The width and height attributes of <img> provide size information for images
  5. The alt attribute of <img> provides an alternate text for an image
  6. The style attribute is used to add styles to an element, such as color, font, size, and more
  7. The lang attribute of the <html> tag declares the language of the Web page
  8. The title attribute defines some extra information about an element
        
        <a href="https://www.example.com">Visit Example Website</a>
        <img src="image.jpg" alt="A sample image">
        <img src="image.jpg" alt="A sample image" width="200" height="150">
        <img src="image.jpg" alt="A beautiful sunset at the beach">
        <p style="color: red; font-size: 18px;">This is a red and large text.</p>
        <html lang="en">
            <!-- Your webpage content here -->
        </html>
        <a href="https://www.example.com" title="Visit Example Website">Link to Example</a>

Tags

Links

Summary

  1. Use the <a> element to define a link
  2. Use the href attribute to define the link address
  3. Use the target attribute to define where to open the linked document
  4. Use the <img> element (inside <a>) to use an image as a link
  5. Use the mailto: scheme inside the href attribute to create a link that opens the user's email program
        
        <a href="https://www.example.com">Visit Example Website</a>
        <a href="https://www.example.com">Visit Example Website</a>
        <a href="https://www.example.com" target="_blank">Visit Example Website (Opens in a new tab)</a>
        <a href="https://www.example.com">
            <img src="image.jpg" alt="An image link to Example Website">
        </a>
        <a href="mailto:example@example.com">Send Email to Example</a>

Tags

  1. <a> - Defines a hyperlink

Images

Image types

  1. APNG - Animated Portable Network Graphics - .apng
  2. GIF - Graphics Interchange Format - .gif
  3. ICO - Microsoft Icon - .ico, .cur
  4. JPEG - Joint Photographic Expert Group image - .jpg, .jpeg, .jfif, .pjpeg, .pjp
  5. PNG - Portable Network Graphics - .png
  6. SVG - Scalable Vector Graphics - .svg

Summary

  1. Use the <img> element to define an image
  2. Use the src attribute to define the URL of the image
  3. Use the alt attribute to define an alternate text for an image, if it cannot be displayed
  4. Use the width and height attributes or the CSS width and height properties to define the size of the image
  5. Use the CSS float property to let the image float to the left or to the right
        
        <img src="image.jpg" alt="A sample image">
        <img src="image.jpg" alt="A sample image">
        <img src="image.jpg" alt="A sample image">
        <img src="image.jpg" alt="A sample image" width="200" height="150">
        <img src="image.jpg" alt="A sample image" style="float: left;">

Head

Summary

  1. The <head> element is a container for metadata (data about data)
  2. The <head> element is placed between the <html> tag and the <body> tag
  3. The <title> element is required and it defines the title of the document
  4. The <style> element is used to define style information for a single document
  5. The <link> tag is most often used to link to external style sheets
  6. The <meta> element is typically used to specify the character set, page description, keywords, author of the document, and viewport settings
  7. The <script> element is used to define client-side JavaScripts
  8. The <base> element specifies the base URL and/or target for all relative URLs in a page

Tags

  1. <head> - Defines information about the document
  2. <title> - Defines the title of a document
  3. <base> - Defines a default address or a default target for all links on a page
  4. <link> - Defines the relationship between a document and an external resource
  5. <meta> - Defines metadata about an HTML document
  6. <script> - Defines a client-side script
  7. <style> - Defines style information for a document

Tables

Tags

  • a table (<table>)
  • a table row (<tr>)
  • a table header (<thead>)
  • a table header cell (<th>)
  • a table body (<tbody>)
  • a table cell (<td>)
  • a table footer (<tfood>)
  • a table footer cell (<td>)
        
        <!-- Start of the table -->
        <table border="1">
          <!-- Table header section -->
          <thead>
            <tr>
              <!-- Table header cell -->
              <th>Header 1</th>
              <th>Header 2</th>
            </tr>
          </thead>

          <!-- Table body section -->
          <tbody>
            <tr>
              <!-- Table cell -->
              <td>Row 1, Cell 1</td>
              <td>Row 1, Cell 2</td>
            </tr>
            <tr>
              <td>Row 2, Cell 1</td>
              <td>Row 2, Cell 2</td>
            </tr>
          </tbody>

          <!-- Table footer section -->
          <tfoot>
            <tr>
              <!-- Table footer cell -->
              <td>Footer 1</td>
              <td>Footer 2</td>
            </tr>
          </tfoot>
        </table>
        <!-- End of the table -->

        <!-- Table to make it responsive -->
        <div style="overflow-x:auto;">
          <table>
            <!-- table content -->
          </table>
        </div>
