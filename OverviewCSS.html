Colours

Summary

In CSS, colors can also be specified using RGB values, HEX values, HSL values, RGBA values, and HSLA values. (140 standart colours)

HEX Value:
In CSS, a color can be specified using a hexadecimal value in the form: #rrggbb
Where rr (red), gg (green) and bb (blue) are hexadecimal values between 00 and ff (same as decimal 0-255).
For example, #ff0000 is displayed as red, because red is set to its highest value (ff) and the others are set to the lowest value (00).
To display black, set all values to 00, like this: #000000.
To display white, set all values to ff, like this: #ffffff.

Text

Summary

  • Text Color: Use the `color` property to set the color of text within an HTML element.
  • Text Alignment: Control the horizontal alignment of text with the `text-align` property.
  • Text Decoration: Add decorative elements to text, such as underlines or overlines, using the `text-decoration` property.
  • Text Transformation: Change the capitalization of text with the `text-transform` property, allowing you to apply uppercase, lowercase, or capitalize the first letter of each word. 
           
               /* CSS for text properties *
               .text-example {
               color: blue; /* Text Color */
               text-align: center; /* Text Alignment */
               text-decoration: underline; /* Text Decoration */
               text-transform: uppercase; /* Text Transformation */
               }

links

Summary

    Link: Concentrate on the section Styling Links only. It nicely summarizes the pseudo-selectors for links.
  • :link - This pseudo-selector is used to target unvisited links (links that the user has not clicked).
  • :visited - Targets visited links (links that the user has clicked).
  • :hover - Styles the link when the mouse cursor hovers over it.
  • :active - Applies styles to the link when it is being activated (e.g., clicked).
  • :focus - Styles the link when it gains focus, such as when navigating with the keyboard. 
            
                /* CSS for styling links */
                  a:link {
                    color: blue; /* Unvisited link color */
                    text-decoration: none; /* Remove underline */
                  }

                  a:visited {
                    color: purple; /* Visited link color */
                  }

                  a:hover {
                    color: red; /* Hover color */
                  }

                  a:active {
                    color: green; /* Active (clicked) link color */
                  }

                  a:focus {
                    outline: 2px solid orange; /* Add focus outline */
                  }

Combinators

Summary

  • **Descendant Combinator (Whitespace):** - Symbol: (space) - Description: Selects elements that are descendants of another element. It matches all instances where the right-hand element is contained within the left-hand element, no matter how deep.
  • **Child Combinator (>):** - Symbol: `>` - Description: Selects elements that are a direct child of another element. It matches elements that are immediately nested within the parent element.
  • **Adjacent Sibling Combinator (+):** - Symbol: `+` - Description: Selects an element that is immediately preceded by another element. It matches the element that shares the same parent and appears directly after the specified element.
  • **General Sibling Combinator (~):** - Symbol: `~` - Description: Selects elements that are siblings of another element and share the same parent. It matches elements that appear anywhere after the specified element. 
            
                /* Examples */
                  div p {
                    /* Selects all <p> elements within a <div> */
                  }

                  ul > li {
                    /* Selects all <li> elements that are direct children of a <ul> */
                  }

                  h2 + p {
                    /* Selects a <p> element that directly follows an <h2> */
                  }

                  h2 ~ p {
                    /* Selects all <p> elements that are siblings of an <h2> */
                  }

Pseudo Classes

Summary

  • Syntax: Pseudo-classes in CSS are used to define special states of elements. They are preceded by a colon (":") and allow for the styling of elements based on their relationships and interaction states.
  • Anchor Pseudo-classes: Pseudo-classes related to anchor (<a>) elements are commonly used to style links. These include :link, which selects unvisited links, and :visited, which selects visited links. Additionally, the :hover pseudo-class is used to style links when they are hovered over by the cursor.
  • CSS - The :first-child Pseudo-class: The :first-child pseudo-class selects the first child element within its parent. It is useful for styling the first occurrence of an element within a container. Pseudo-classes provide a powerful way to apply styles to elements based on their relationships and user interaction, allowing for dynamic and targeted styling of web content.

Parent and Child

Summary

Parent Element:
A parent element is an HTML element that contains one or more child elements. In the HTML document's structure, parent elements enclose or "contain" child elements. For example, a <div> element containing several <p> elements makes the <div> the parent element of those <p> elements.
Child Element:
A child element is an HTML element that is contained within a parent element. Child elements are nested within their parent elements. For example, within a <ul> (unordered list) element, each <li> (list item) is a child element of the <ul>.
