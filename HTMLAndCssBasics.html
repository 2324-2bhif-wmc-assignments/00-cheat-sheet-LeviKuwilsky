HTML:

  • Name of Elements: HTML elements are usually represented by a name enclosed in angle brackets, such as <element>.
  • Void Elements: These are elements that don't have a closing tag. They are denoted by using a forward slash before the closing angle bracket, like <img src="image.jpg"/>.
  • Attributes: HTML elements can have attributes that provide additional information about the element. Attributes are specified within the opening tag and have a name and a value, separated by an equal sign. For example, in <a href="https://example.com">, href is an attribute.
  • DOCTYPE Declaration for HTML5: The correct DOCTYPE declaration for HTML5 is: <!DOCTYPE html>.
  • Syntactically Correct Statements:
    • <article>: <article></article>
    • <body>: <body></body>
    • Line break (<br/>): <br/>
    • Headings (<h1>, ...): <h1>This is a Heading</h1>
    • Section with meta-information (<head>): <head><meta charset="UTF-8"></head>
    • Top level element (<html>): <html></html>
    • Image named Team.jpg (<img src="Team.jpg">): <img src="Team.jpg" alt="Team Image">
    • Paragraph (<p>): <p>This is a paragraph.</p>
    • Section (<section>): <section>This is a section.</section>
    • Label appearing in the title bar (<title>): <title>Page Title</title>
    • Link to other pages (<a>): <a href="https://example.com">Visit Example</a>
    • Comment: <!-- This is a comment -->
    • Head with title and meta elements: <head> <title>Page Title</title> <meta charset="UTF-8"> </head>

CSS:

  • CSS Rule Components:
    • Selector: The selector specifies which HTML elements the rule applies to.
    • Property: The property defines the CSS property you want to set.
    • Value: The value specifies the value for the CSS property.
  • Combinators:
    • Direct Child: parent > child
    • Descendant: ancestor descendant
    • Adjacent Sibling: element + adjacent-element
    • Sibling: element ~ sibling-element
  • Colors:
    • Color Names: Example: color: red;
    • Hexadecimal Notation: Example: color: #FF5733;
  • Declarations:
    • Color: color: blue;
    • Background Color: background-color: #EFEFEF;
    • Pseudo Classes for <a> Element: a:hover { ... }
    • Background Image: background-image: url('image.jpg');
    • Text Alignment: text-align: center;
    • Text Decoration: text-decoration: underline;
    • Text Transformation: text-transform: uppercase;
  • Font Styles:
    • Serif Fonts: Fonts with decorative strokes at the ends of characters.
    • Sans-serif Fonts: Fonts without decorative strokes.
  • Font Declarations:
    • Font Families: font-family: Arial, sans-serif;
    • Font Style: font-style: italic;
    • Font Sizes: font-size: 16px;
    • Font Weights: font-weight: bold;
